# video_player
※ビルドするためにenvファイルが必要です。

## 機能
* Youtubeの動画再生を行えます。
* バックグラウンドでの再生が行えます。
* 音声再生位置をローカルに保存し、再度再生を行う際は保存した位置から再生できます。
